# Playwright CLI Strategy Deck

This repository hosts a static GitHub Pages site for the Playwright CLI leadership deck.

## Live URL

- **GitHub Pages:** https://arvind3.github.io/Playwrite-CLI/

## Notes

- The site is deployed automatically from `main` using GitHub Actions.
- If the URL is unavailable, ensure:
  1. The repo default branch is `main`.
  2. GitHub Pages is enabled and set to **GitHub Actions** as the source.
  3. The latest `Deploy static site to GitHub Pages` workflow run completed successfully.
